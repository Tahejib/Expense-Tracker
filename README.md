# Expense Tracker

This is a web application to help you track your daily expenses easily.  
You can add, edit, and delete expenses and see visual summaries with charts.

## Features
- Add new expenses with category and amount
- Edit or delete existing expenses
- View summary of expenses by category
- See charts for better expense visualization

## How to run
1. Open the project in your IDE
2. Set up your database connection (MySQL or your choice)
3. Run the Spring Boot application
4. Open your browser and go to `http://localhost:8080`
